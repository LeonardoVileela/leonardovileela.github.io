  1. Introdução

Este manual tem como finalidade orientar os usuários na utilização das funcionalidades do Sistema “Gerenciador Pericial”, facilitando assim o acesso e entendimento do sistema. Este manual descreve todos os procedimentos necessários para a inicialização da solução, contendo uma descrição detalhada e imagens de cada atividade e seus respectivos passos.  

O Gerenciador Pericial consiste em um software que pretende solucionar o problema de gerenciamento e distribuição de laudos da perícia. Os usuários são peritos criminais, agentes de polícia científica, médicos legistas, peritos papiloscopista e estagiários do município de Paranaíba - Mato Grosso do Sul. O sistema será responsável pelo gerenciamento de usuários, pela distribuição e registros de entrada e saída de laudos e também pela visualização de relatórios mensais.

  1. Acesso ao sistema

  1. Cadastro de Usuário

O primeiro passo para conseguir acesso ao sistema e suas funcionalidades é realizar o cadastro. O usuário deve acessar a tela de login e clicar na opção “Não possui conta? Cadastre-se.” situada logo abaixo do campo onde o e-mail é informado.

 

Imagem 1 - Acessando a tela de cadastro.

Logo após clicar na opção acima, o usuário será redirecionado para uma tela de cadastro, onde ele deverá informar dados como nome completo, telefone, e-mail, cargo, senha e confirmação de senha.

Assim que informar todos os dados solicitados, o usuário deverá clicar no botão “Cadastrar” e aguardar até que algum administrador confirme seu acesso ao sistema.

Imagem 2 - Tela de cadastro de usuário.

  1. Login

Para realizar o login é necessário que o usuário informe nos campos o e-mail e senha cadastrados. Após o preenchimento, deverá clicar no botão “Entrar”, destacado na imagem abaixo.

Imagem 3 - Tela de Login.

Após clicar no botão “Entrar”, se o usuário já possuir acesso à plataforma, será redirecionado para a tela inicial do sistema.

Caso o usuário perca sua senha, será possível realizar a recuperação e alteração da mesma clicando na opção “Esqueceu sua senha? Clique aqui para recebê-la por e-mail.”, situada abaixo do campo “senha” na tela de login, como é possível observar na imagem abaixo.

 

Imagem 4 - Tela de login destacando a recuperação de senha.

Após clicar na opção acima, o usuário será redirecionado para uma página de recuperação de senha, onde ele deverá informar o e-mail que foi cadastrado no sistema. Após o preenchimento, o usuário deverá clicar no botão “recuperar senha”.

Após clicar no botão ilustrado acima, o usuário receberá em seu e-mail cadastrado um link para redefinição de senha. Quando clicar no link enviado para o e-mail, será redirecionado para a tela abaixo, então o usuário deve digitar sua nova senha e depois digitar novamente no campo abaixo para que ocorra a confirmação.
